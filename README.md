# WWYD Bot

Inspired by the [UW Riichi Club bot](https://github.com/sunmingyang0115/riichinya-v2), this is a bot that aims to
replicate the WWYD functionality.

## Commands

* `/wwyd random`: Generates a random WWYD for practice
* `/wwyd daily leaderboard`: Guild leaderboard
* `/wwyd daily score`: Guild score
* `/wwyd daily force`: Forces the bot to send a daily WWYD in the channel (Requires Manage-Channels Permission)
* `/wwyd daily toggle`: Toggles wwyd daily for the guild (Requires Manage-Channels Permission)
