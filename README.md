# WWYD Discord Bot

This is a bot that sends out WWYD quizzes every day at 10am ET for members to complete.

Massive thanks to the folks at UW Riichi Club for inspiring this project.

## Commands

* `/wwyd random`: Generates a random WWYD for practice
* `/wwyd daily leaderboard`: Guild leaderboard
* `/wwyd daily score`: Guild score
* `/wwyd daily toggle`: Toggles wwyd daily for the channel (Requires Manage-Channels Permission)
* `/wwyd daily force`: Forces the bot to send a daily WWYD in the channel (Requires Manage-Channels Permission)
